Script de automação para Telegram

• Conecta-se à sua conta do Telegram usando seu api_id and api_hash

• Monitora um chat de origem (source_chat) de forma contínua.

• Verifica as mensagens nesse chat e, se uma delas contiver qualquer tipo de mídia (foto, vídeo, documento, etc.), ele a encaminha.

• Encaminha a mensagem com mídia para um chat de destino (target_chat).

• Ignora as mensagens que são apenas texto.
